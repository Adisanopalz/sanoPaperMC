🎮 Sano PaperMC

Sano PaperMC adalah aplikasi web modern, ringan, dan super cepat untuk mengunduh file server.jar PaperMC. Dibangun dengan fokus pada User Experience (UX) dan antarmuka yang memanjakan mata, web ini memungkinkan admin server Minecraft untuk mencari, memfilter, dan mengunduh versi PaperMC yang mereka butuhkan hanya dengan satu klik.

✨ Fitur Utama

⚡ Unduhan Instan: Temukan dan unduh versi yang Anda inginkan (dari versi terbaru hingga versi lawas seperti 1.8.8) tanpa harus melewati banyak halaman bertele-tele.

🎨 Custom Color Theme: Sesuaikan warna aksen web sesuka hati Anda menggunakan pemilih warna (color picker) bawaan. Seluruh elemen UI akan beradaptasi secara dinamis!

🌗 Dark & Light Mode: Dukungan penuh untuk tema gelap dan terang yang nyaman di mata, lengkap dengan animasi transisi fluid.

🌐 Multi-bahasa (i18n): Dapat dialihbahasakan antara Bahasa Indonesia dan English secara real-time.

🔍 Pencarian & Filter Cerdas: Menyediakan sticky bar untuk memfilter versi berdasarkan versi mayor (misal: 1.20, 1.21) atau mencari versi spesifik melalui kolom pencarian.

📱 Desain Responsif & Animatif: Menggunakan gaya glassmorphism (efek kaca buram) modern dengan animasi mikro saat elemen di-hover, optimal di layar desktop maupun perangkat seluler.

🛠️ Teknologi yang Digunakan

React - Library JavaScript untuk membangun antarmuka pengguna.

Tailwind CSS - Utility-first CSS framework untuk styling yang cepat dan responsif.

Lucide React - Kumpulan ikon SVG cantik dan konsisten.

🚀 Cara Menjalankan (Development)

Prasyarat: Pastikan Anda sudah menginstal Node.js di sistem Anda.

Clone repositori ini ke komputer Anda:

git clone [https://github.com/username/sano-papermc.git](https://github.com/username/sano-papermc.git)


Masuk ke direktori proyek:

cd sano-papermc


Install semua dependensi yang dibutuhkan:

npm install
# atau jika menggunakan yarn:
yarn install


Jalankan development server:

npm run dev
# atau
yarn dev


Buka http://localhost:5173 (atau port lain yang disediakan) di browser Anda.

📦 Sumber Data

Data tautan unduhan dan versi di-generate secara statis mengarah langsung ke penyimpanan PaperMC (fill-data.papermc.io). File server.jar langsung diunduh dari source asli untuk memastikan keaslian dan keamanan file.
